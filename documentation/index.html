    • Introduction

    Thank you for choosing my Template. Amscen is an HTML responsive website template created for corporate and formal kinds of busines websites. The template is not only created for any specific category of corporate businesses, but it can be use for any businesses who needs a website with formal layout and structure. The template has many features where the performance optimization and customization easiness are the major advantages. Template is not included any larger frontend frameworks or libraries. So, anyone who has basic knowledge of HTML, SASS, CSS & JS can be edit the template more easily.

    Features

    The template has many features which allow you to modify and customize the template based on your ideas and requirements. Followings are the major features.

    • Every part of the Template is Customizable
    • Cross Browser compatible Chrome, FireFox, Opera, Edge, Brave etc.
    • W3 Validated HTML Markup
    • W3 Validated CSS
    • Responsive Design with Mobile First Approach. If you are not familiar with it, please refer it here. Mobile First Approach
    • No bigger JavaScript Libraries
    • No bigger CSS Frameworks
    • Used native JavaScript and CSS for all user interphace designs
    • Template is Lightweight because not used numarous JS and CSS dependencies. Every page includes only one JS and CSS dependency.
    • CSS created with BEM Methodology. If you are not familiar with it, please refer it here BEM Methodology
    • Performance Optimized
    • All image files are in webp format to improve performance

    JavaScript knowledge is required to manage the script part of the website. If you are not familiar with JavaScript, you can learn it here: Learn JavaScript

    Getting Started

    Unzip the folder you downloaded. You can see two folders after unzipping the folder. One is HTML and another is Documentation where the HTML folder is the main folder that you need to upload to the server. You can start to modify and use template once you are meet the above said prerequisites. It is not mandatory that you need an expert or intermediate level experience. The prerequisites are means that a basic knowledge is also enough to customize the template. Followings are the explanation of folder structure and role of each dependency files in the template.

    Visual Customization

    Every part of the Template is customizable as per your requirements. Followings are the explanation of various sections of websites and files which is related to the template customization. Before customization make sure that you are watching the SASS folder using any of the SASS compilers and it is compiled to the CSS folder.

    Color Theme

    All the colors in the template is defined using SASS variables. The Hex value colors are stored in a SASS Variable and used everywhere. Following is the SASS variable file to modify

    amscen/html/assets/sass/abstracts/_variables.scss

    Typography

    Following is the SASS partial file with typography details which is also covered in the Fonts section. In the following partial file you can change the Typography size, color etc.

    amscen/html/assets/sass/base/_typography.scss

    Icons

    All Icons used in the template is SVG format. The color an SVG canot be assigned with CSS. Colors of an SVG is defined inside the SVG file. So you need to open the SVG file in the editor and change the value in the code as follows.

    <svg><path fill="ff0000"></svg>

    Favicon

    The template is used sample favicon which you can make it the logo of your company. Upload a PNG or JPG file of 16x16 resolution and generate favicon in the following link. Favicon Generator

    Put the favion in the image folder as follows

    amscen/html/assets/images

    Favicon is defined in the "head" section of the template as follows

    <link rel="icon" type="image/x-icon" href="favicon.ico">

    The JavaScript behaviour of the Hero Slider can be manage in the following JS file. Native SwiperJS JS file is included in it. In the following JS file you can manage the slider speed, delay etc.

    amscen/html/assets/js/components/hero-slider.js
